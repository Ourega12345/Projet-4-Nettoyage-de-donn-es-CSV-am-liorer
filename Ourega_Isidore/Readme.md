## Description des classes

###  GestionnaireFichier

Gère les opérations liées aux fichiers : sélection du fichier CSV, lecture des données et sauvegarde du fichier nettoyé.

### NettoyeurDonnees

Contient la logique de nettoyage des données : suppression des espaces, conversion en majuscules et gestion des valeurs manquantes.

### App

Point d’entrée du programme. Coordonne les différentes étapes : sélection du fichier, lecture, nettoyage des données et sauvegarde du résultat.
